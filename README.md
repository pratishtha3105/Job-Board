# Job-Board

This project is a full stack JavaScript application designed to fetch, filter, and display entry-level software job postings from various job board APIs. The project demonstrates the use of React for the front end, Node.jsand Express for the back end, and Redis for data storage. The application is deployed using NGINX on DigitalOcean with SSL for security.

# Features

Fetches job postings from GitHub Jobs API

Filters jobs to display only entry-level software jobs

React front end with Material UI for responsive design

Node.jsand Express back end for API handling

Redis for caching and storing filtered jobs

Deployment using NGINX on DigitalOcean with SSL
